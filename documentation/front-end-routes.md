# Frontend Routes

* Root
  * App
    * NavBar
    * Main content
    * Footer

The following routes will render components between NavBar and Footer

* /
  * Home Page
* /login
  * LoginForm
* /projects/:id
  * ProjectBox
    * ProjectDetails
* /project/messageBoard/:id
  * MessageBoard
  * Message
    * CommentsComponent
* /project/to-do
  * ToDoComponent
    * ToDoDetails
* /users/:id
  * ProfileComponent
